# bootstrap-web-admin
Bootstrap admin templates available on web

#### Basic setup commands
```shell
    $ git clone https://github.com/suthaharan/bootstrap-web-admin.git yourfoldername
    $ cd yourfoldername
    $ npm install
    $ bower install
    $ npm run buildrtl [name of the build folder]
    $ gulp serve
```
